## Manual Installation

### Installation dependency

```codeBlock bash
pnpm add @easy-shadcn/react @easy-shadcn/utils
```

### Config tailwindcss

参考 [shadcn/ui Manual Installation](https://ui.shadcn.com/docs/installation/manual).
1. Add Tailwind CSS
2. Configure path aliases
3. Configure tailwind.config.js
4. Configure styles

### Msodify tailwind.config.js

add `'./node_modules/@easy-shadcn/react/dist/**/*.{js,ts,jsx,tsx}'` to content

```js {7}
// tailwind.config
import type { Config } from "tailwindcss";

const config: Config = {
  content: [
    // ...
    './node_modules/@easy-shadcn/react/dist/**/*.{js,ts,jsx,tsx}'
  ],
  // ....
};
// ...
```
