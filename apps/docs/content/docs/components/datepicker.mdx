## API


| Attribute       | Type                                | Description      | Default    |
| :-------------- | :---------------------------------- | :--------------- | :--------- |
| mode            | `'single' \| 'multiple' \| 'range'` | Datepicker Mode  | `'single'` |
| placeholder     | `string`                            | Default text     | -          |
| buttonClassName | `string`                            | Button classname | -          |
| dateFormat      | `string`                            | see [date-fns format](https://date-fns.org/v3.6.0/docs/format) | -          |
