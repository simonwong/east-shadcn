## Usage

```tsx
import { Popover } from "@easy-shadcn/react"

<Popover
  content={"Popover Content"}
>
  Popover
</Popover>
```

## Examples

### Basic

<CodeDemo code={`
const Demo = () => {
  return (
    <Popover
      content={"Popover Content"}
    >
      <Button>Show</Button>
    </Popover>
  )
}
render(<Demo />)
`}
/>

## API

| Attribute      | Type                    | Description                  | Default     |
| :------------- | :---------------------- | :--------------------------- | :---------- |
| content        | `React.ReactNode`       | Popover Content              | -           |
| contentProps   | `PopoverContentProps`   | Popover Content Props        | -           |
